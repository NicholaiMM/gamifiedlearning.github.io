# Gamified Learning v14.0 | Stellara

> 🎮 Mobile-first educational gaming platform featuring the Stellara Match-3 puzzle game

![Version](https://img.shields.io/badge/version-14.0.0-blue)
![Status](https://img.shields.io/badge/status-production-green)

## Features

### 🌟 Stellara Match-3 Game
- 8x8 grid puzzle gameplay with 6 unique tile types
- Smooth animations and responsive controls
- Score tracking with XP rewards

### 🗺️ Overworld Map
- Interactive node-based navigation
- Tampa HQ, Stellara Grid, Neon Leap, Seven Stars, Manila B06
- Progress tracking and unlock system

### 🛡️ Brand Vault
- Logo concept gallery (Gold Classic, Modern Blue/Silver, Pixel Art)
- DevOps tool integration links
- Resource management

### 💻 Terminal Interface
- Command-line style interaction
- Commands: `help`, `stats`, `map`, `bio`, `clear`, `brand`

## Tech Stack

- **Frontend**: React 18 with JSX
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Backend**: Firebase (Auth & Firestore)
- **PWA**: Progressive Web App ready

## Quick Start

1. Clone the repository
2. Open `index.html` in a browser (or serve with a local server)
3. For production, deploy to GitHub Pages

```bash
# Local development server
python -m http.server 8080
# Then open http://localhost:8080
```
